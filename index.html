Lesson 1: Variables and Printing

Variables are used to store data. In Lua, use the local keyword to create a variable that only exists inside the current block.

local name = "Lua"
print("Hello, " .. name)

Explanation: The .. operator joins strings together (called concatenation).

Variable: A storage location that holds a value that can change while the program runs.
Concatenation: Combining two or more strings into one using ...

Lesson 2: Tables (Lua‚Äôs Arrays & Dictionaries)

Tables are Lua‚Äôs only data structure ‚Äî they can act as lists, dictionaries, or even objects.

local fruits = {"apple", "banana", "cherry"}
print(fruits[1])  -- prints "apple"

local person = {name = "Alex", age = 20}
print(person.name)

Tip: Lua starts indexing at 1, not 0!

Table: A collection type used to store multiple values (lists or key-value pairs).

Lesson 3: Loops and Functions

Loops allow you to repeat code, while functions help organize it for reuse.

for i = 1, 5 do
    print("Count: " .. i)
end

function greet(name)
    print("Hello, " .. name)
end

greet("Lua Learner")
Function: A block of reusable code that performs a specific task.

Lesson 4: Conditionals (If, Else, and Logic)

Conditionals let your program make decisions based on logic.

local age = 18

if age >= 18 then
    print("You are an adult!")
else
    print("You are a minor.")
end
Conditional: Code that executes only when certain conditions are true.

Lesson 5: Roblox Lua Basics

Roblox uses a version of Lua called Luau for scripting game mechanics.

local part = Instance.new("Part")
part.Size = Vector3.new(4, 1, 4)
part.Position = Vector3.new(0, 5, 0)
part.Anchored = true
part.Parent = workspace

This code creates a floating block in Roblox Studio!

Instance: A Roblox object, like a Part, Script, or GUI element.
