Lesson 1: Variables and Printing

In Lua, you can store data in variables using the local keyword.

local name = "Lua"
print("Hello, " .. name)

Explanation: The .. operator joins strings together.

Lesson 2: Tables (Lua’s Arrays & Dictionaries)

Tables are Lua’s main data structure. You can use them as lists or maps.

local fruits = {"apple", "banana", "cherry"}
print(fruits[1])  -- prints "apple"

local person = {name = "Alex", age = 20}
print(person.name)

Tip: Lua tables start counting at 1, not 0!

Lesson 3: Loops and Functions

Loops let you repeat actions. Functions let you organize code.

for i = 1, 5 do
    print("Count: " .. i)
end

function greet(name)
    print("Hello, " .. name)
end

greet("Lua Learner")
